С описанием протоколов у меня проблемы

Использует основной протокол и формат, такие как HTTP
Данные передаются в фотмате JSON

Базовый адрес сервиса - http://localhost:19812/
Метод получения статуса сервера - api/Status/PlanWorkTime
Полный путь для получения статуса сервера - http://localhost:19812/api/Status/PlanWorkTime

Пример:

{
	"state":1,           							Состояние сервера (1 - активен, 0 - не активен)
	"planWorkTime":      							Планируемы даты отключения сервера (массив)
	[
		{
			"beginTime":"2016-07-18T13:44:32",		Дата начала технической работы на сервере (формат даты: yyyy-MM-ddTHH:mm:ss)
			"endTime":"2016-07-18T13:56:32"			Дата окончания технической работы на сервере (формат даты: yyyy-MM-ddTHH:mm:ss)
		},
		{
			"beginTime":"2016-07-18T13:46:21",      Дата начала технической работы на сервере
			"endTime":"2016-07-18T13:57:21"			Дата окончания технической работы на сервере
		}
	]
}
